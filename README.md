#  Project Overview  

This project demonstrates the implementation of a **Singly Linked List** using C programming.  
It performs basic operations such as **insertion, deletion, and traversal** using a dynamic data structure.



#  Objectives  

- To understand the concept of linked list data structure  
- To implement dynamic memory allocation in C  
- To perform basic operations on linked list  
- To design a menu-driven program for user interaction  



#  Features  

- Insert a node into the linked list  
- Delete a node from the linked list  
- Traverse and display the linked list  
- Dynamic memory allocation using `malloc`  
- Menu-driven interface  


# Concepts Used  

- Structures in C  
- Pointers  
- Dynamic Memory Allocation (`malloc`, `free`)  
- Linked List Operations  
- Control Statements
